# cpp-basic

This repository contains C++ programs written for learning and revision.

Topics covered:
- Basic input/output and control structures
- Functions (inline, default arguments)
- Object Oriented Programming
  - constructors and destructors
  - static members
  - inheritance and polymorphism
  - operator overloading
- File handling
- Dynamic memory allocation

Some programs are adapted from college lab exercises and rewritten
with clean structure and naming for practice.
